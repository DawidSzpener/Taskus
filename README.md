# Tasker

Android and IOS application to help me with structuring my day efficently. Save daily tasks, notes and complete goals for the day!

## Getting Started

For local development:

```
$ git clone https://github.com/DawidSzpener/easy-app
$ npm install
$ npm start
```

## Running the tests

### Code style

To run the linter

```
$ yarn run lint
```
